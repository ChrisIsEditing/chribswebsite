A FOREVER MINECRAFT SMP

Server Information

ChribsMC is a Forever-SMP Minecraft Server running on the latest Minecraft Java-Edtion versions. For whitelisting, please contact Chribs.

FEATURES

Waypoints

A simple waypoint plugin.

Basic Commands:

/wsave "Name of Place" - Save a waypoint

/wtp "Name of Place" - Teleport to selected waypoint

SkinsRestorer

Easily change your skin in the server.

Basic Commands:

/skin set [Link to Skin's Image] - Set your skin

Other Plugins:

Other plugins used, which dont support/need user commands.

Basic Commands:

ViaVersion - ViaVersion allows players on older or newer clients to join the server.


Boundless Forging - Removes "Too Expensive" warning in Anvils.


Villager in a Bucket - Lets players pick up villagers in buckets.


Backuper - The server is frequently backed up! :D


Floodgate/Geyser - Lets Bedrock players join too.


SquareMap - A live 2D Map of the Server, which can be accesed via Web Browser.


Teleport Cost - Any teleports will cost you XP! [0.5 Levels]
